# CineLog Frontend

Standalone React + Vite frontend for the CineLog backend.

## Setup

1. Copy the env template:

```bash
cp .env.example .env
```

2. Set:

- `VITE_API_BASE_URL`
- `VITE_GOOGLE_CLIENT_ID`

3. Install dependencies:

```bash
npm install
```

4. Start the app:

```bash
npm run dev
```

The app expects the Spring Boot backend to be running separately, typically on `http://localhost:8080`.
